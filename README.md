Ecommerce Website

A full-stack e-commerce web application built as a learning project for web development practice

Features
User authentication with login and registration

Product browsing and catalog display

Shopping cart functionality

Order management system

Responsive design for mobile and desktop

Tech Stack
Backend: Python with Flask framework

Frontend: HTML, CSS, JavaScript

Database: SQLite or PostgreSQL for product and user data

Deployment: AWS cloud services



Future Enhancements
Payment integration (Stripe/PayPal)

Admin dashboard

OTP verification for authentication

Product search and filters
